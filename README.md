## StereoPay Media Manager

A media manager

## Description

A media manager for audio and images

## Built With 🌩️

- NestJS
- MySQL

## Getting Started ☀️

To get a local copy up and running follow these simple example steps.

### Prerequisites

Install Node and npm using the `npm install` command on your command line, then install yarn using `npm install --global yarn`.

### Setup

Clone this repository by typing `git clone https://github.com/ikechukwu-peter/nip-api.git` on your terminal.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
